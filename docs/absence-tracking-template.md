# Absence Tracking Template

Use this when a character drops out, fades into the wallpaper, or returns after a gap that the draft has not earned properly.

This is especially useful for:
- campaigns being adapted into prose
- ensemble stories with rotating focus
- stories affected by player absence, scheduling gaps, or skipped sessions
- narratives where a return is meant to feel loaded rather than administrative

## Purpose

An absence tracker stops character continuity from becoming accidental bookkeeping.

It helps the audit track:
- when a character is absent
- whether the absence is acknowledged in-world
- how other people react to it
- what changes because they are gone
- how the eventual return lands

Without this, a disappearance that was accidental in play often reads accidental in fiction too.

## How to use it

Create one row for each meaningful stretch of absence or weak-presence drift.

A row can represent:
- a full absence across one or more scenes or sessions
- a character being mentioned but not present
- a return after a meaningful gap
- a character being physically present but functionally absent from the story logic

Use the tracker before prose adaptation converts table logistics into invisible continuity damage.

## Minimal version, if the material is rough

If the source is messy, track only:
- who vanished
- when they last mattered clearly
- whether the story notices they are gone
- what must be true by the time they return
- whether the return currently feels like payoff or admin

## Common failure modes this catches

### 1. Character dropouts read as forgetfulness

The draft never tells the reader whether the gap is intentional, worrying, or strategically meaningful.

### 2. Group dynamics do not change after an absence

A missing character should alter plans, trust, labour, mood, or power. If nothing changes, the cast may feel weightless.

### 3. Returns land flat

A return should be legible as relief, suspicion, payoff, disruption, or betrayal. If it just updates the headcount, the story wastes the gap.

### 4. Physically present, narratively absent characters

Someone appears in scenes but exerts no pressure on decisions, emotion, or continuity, which can be as damaging as literal disappearance.

## Audit prompts to pair with the tracker

Ask these while filling it out:
- Does the story notice this person is gone?
- Who should care most, and how would that show up?
- What becomes harder, riskier, or emotionally different because of the absence?
- Is the return being treated as payoff, suspicion, relief, or mere logistics?
- Would a reader interpret this gap as intentional if they did not know the table history?

This is one of the fastest ways to prove the continuity-audit offer understands campaign adaptation rather than merely tolerating it.

Character absence is where table logistics most often leak through the walls of fiction.

A clean absence tracker turns that leak into a controllable structural choice.
